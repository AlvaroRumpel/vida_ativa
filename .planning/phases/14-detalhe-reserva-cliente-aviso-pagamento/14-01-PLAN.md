In `my_bookings_screen.dart`, update every `BookingCard` rendering to be wrapped in an `InkWell` (or `GestureDetector`) that calls `_showDetailSheet(context, b, isFuture)`.

Add a helper method `_showDetailSheet`:
```dart
void _showDetailSheet(BuildContext context, BookingModel booking, bool isFuture) {
  final bookingCubit = context.read<BookingCubit>();
  showModalBottomSheet(
    context: context,
    isScrollControlled: true,
    shape: const RoundedRectangleBorder(
      borderRadius: BorderRadius.vertical(top: Radius.circular(20)),
    ),
    builder: (_) => ClientBookingDetailSheet(
      booking: booking,
      bookingCubit: bookingCubit,
      isFuture: isFuture,
    ),
  );
}
```

Note: `bookingCubit` is captured before `showModalBottomSheet` builder (same pattern as Phase 4 and Phase 8 decisions — bottom sheet subtree has no BlocProvider access).
