**4. Create `lib/features/booking/cubit/booking_cubit.dart`:**

Class `BookingCubit extends Cubit<BookingState>` with:

Constructor parameters: `required FirebaseFirestore firestore`, `required String userId`.
Fields: `final FirebaseFirestore _firestore`, `final String _userId`, `StreamSubscription<QuerySnapshot>? _sub`.
Constructor body calls `_startStream()`. Initial state: `const BookingInitial()`.

**Method `_startStream()`:**
- Emit `const BookingLoading()`
- Subscribe to `_firestore.collection('bookings').where('userId', isEqualTo: _userId).snapshots()`
- No `.orderBy()` — sort locally in Dart to avoid composite index requirement
- On data: map docs via `BookingModel.fromFirestore(d)`, emit `BookingLoaded(bookings)`
- On error: emit `const BookingError('Erro ao carregar reservas.')`
- Store subscription in `_sub`

**Method `Future<void> bookSlot({required String slotId, required String dateString, required double price, required String startTime})`:**
- Generate doc ID: `BookingModel.generateId(slotId, dateString)`
- Get doc ref: `_firestore.collection('bookings').doc(docId)`
- Call `_firestore.runTransaction((tx) async { ... })`
- Inside transaction:
  - `final snap = await tx.get(ref);`
  - If `snap.exists`: `final existing = BookingModel.fromFirestore(snap); if (!existing.isCancelled) throw Exception('slot_already_booked');`
  - Create BookingModel: `BookingModel(id: docId, slotId: slotId, date: dateString, userId: _userId, status: 'pending', createdAt: DateTime.now(), startTime: startTime, price: price)`
  - `tx.set(ref, booking.toFirestore());`
- This method does NOT emit any cubit state — the stream subscription picks up the new booking reactively
- Throws on error (caller handles)

**Method `Future<void> cancelBooking(String bookingId)`:**
- `await _firestore.collection('bookings').doc(bookingId).update({'status': 'cancelled', 'cancelledAt': Timestamp.fromDate(DateTime.now())});`
- Does NOT emit any cubit state — stream picks up change reactively
- Throws on error (caller handles)

**Override `Future<void> close()`:** Cancel `_sub`, call `super.close()`.

Imports: `dart:async`, `package:cloud_firestore/cloud_firestore.dart`, `package:flutter_bloc/flutter_bloc.dart`, booking_model, booking_state.
