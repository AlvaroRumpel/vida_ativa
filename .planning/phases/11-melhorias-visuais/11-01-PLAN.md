**Step 3 — Create `lib/features/schedule/ui/slot_day_view.dart`:**

A StatefulWidget `SlotDayView` that wraps the `calendar_view` `DayView<SlotViewModel>`.

Constructor parameters:
- `ScheduleState state` (required)
- `DateTime selectedDay` (required)

State management:
- `late EventController<SlotViewModel> _eventController` created in `initState()`, disposed in `dispose()`.
- `late ScrollController _scrollController` created in `initState()`, disposed in `dispose()`.

Method `_updateEvents(List<SlotViewModel> slots, DateTime day)`:
- Call `_eventController.removeWhere((_) => true)` to clear all events.
  - If `removeWhere` does not exist in 2.0.0 API, fallback: recreate `_eventController` and call `setState` to rebuild DayView.
- For each `vm` in `slots`:
  - Parse `vm.slot.startTime` (format "HH:mm") into hour and minute.
  - Create `DateTime start = DateTime(day.year, day.month, day.day, hour, minute)`.
  - Create `DateTime end = start.add(const Duration(hours: 1))`.
  - Add `CalendarEventData<SlotViewModel>(date: day, startTime: start, endTime: end, title: vm.slot.startTime, color: _colorForStatus(vm.status), event: vm)`.

Color helper `_colorForStatus(SlotStatus status)`:
```dart
Color _colorForStatus(SlotStatus status) => switch (status) {
  SlotStatus.available  => AppTheme.primaryGreen.withOpacity(0.2),
  SlotStatus.booked     => Colors.grey.shade200,
  SlotStatus.myBooking  => AppTheme.primaryGreen,
  SlotStatus.blocked    => const Color(0xFFE53935).withOpacity(0.2),
};
```

Dynamic hour range methods:
- `_computeStartHour(List<SlotViewModel> slots)`: if empty return 6; else `(minHour - 1).clamp(0, 23)` where `minHour = slots.map(vm => int.parse(vm.slot.startTime.split(':')[0])).reduce(min)`.
- `_computeEndHour(List<SlotViewModel> slots)`: if empty return 22; else `(maxEndHour + 1).clamp(1, 24)` where `maxEndHour = slots.map(vm => int.parse(vm.slot.startTime.split(':')[0]) + 1).reduce(max)`.
  Import `dart:math` for `min`/`max`.

`build()` method:
- Use sealed class exhaustive switch on `state`:
  - `ScheduleInitial` / `ScheduleLoading`: return a shimmer loading widget — adapt SlotSkeleton pattern: `StatefulWidget` with `AnimationController(duration: 800ms)`, `Tween(0.3..0.8)`, repeat reverse. Render 3 grey `Container` blocks at simulated positions (heights ~60px each, spaced vertically). Can be a private `_TimelineSkeleton` widget inside this file.
  - `ScheduleError`: return `Center(child: Text(message, style: TextStyle(color: Colors.red)))`.
  - `ScheduleLoaded` when `isBlocked`: return `Center(child: Padding(padding: EdgeInsets.all(24), child: Text('Dia bloqueado \u2014 sem horarios disponiveis.', textAlign: TextAlign.center)))`.
  - `ScheduleLoaded` when `slots.isEmpty`: return `Center(child: Padding(padding: EdgeInsets.all(24), child: Text('Nenhum horario disponivel para este dia.', textAlign: TextAlign.center)))`.
  - `ScheduleLoaded` with slots: call `_updateEvents(slots, selectedDay)`, compute `startHour`/`endHour`, then return `DayView<SlotViewModel>(...)`.

DayView configuration:
```dart
DayView<SlotViewModel>(
  key: ValueKey(selectedDay),  // force rebuild on day change
  controller: _eventController,
  initialDay: selectedDay,
  startHour: startHour,
  endHour: endHour,
  heightPerMinute: 1.0,
  showLiveTimeLineInAllDays: true,
  backgroundColor: Colors.white,
  dayTitleBuilder: (_) => const SizedBox.shrink(),
  timeStringBuilder: (dt, {secondaryDate}) =>
      '${dt.hour.toString().padLeft(2, '0')}:${dt.minute.toString().padLeft(2, '0')}',
  eventTileBuilder: (date, events, boundary, startDuration, endDuration) {
    if (events.isEmpty) return const SizedBox.shrink();
    final vm = events.first.event!;
    return SlotEventTile(
      viewModel: vm,
      onTap: vm.status == SlotStatus.available
          ? () => _showBookingSheet(vm)
          : null,
    );
  },
  onEventTap: (events, date) {
    if (events.isEmpty) return;
    final vm = events.first.event as SlotViewModel?;
    if (vm == null || vm.status != SlotStatus.available) return;
    _showBookingSheet(vm);
  },
)
```
