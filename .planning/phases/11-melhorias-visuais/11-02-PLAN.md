```dart
/// Spacing tokens for consistent padding and margin values.
///
/// Usage: `EdgeInsets.all(AppSpacing.md)` or `SizedBox(height: AppSpacing.sm)`
class AppSpacing {
  AppSpacing._(); // Private constructor — use static members only

  /// 4.0 — Icon gaps, badge internal padding, color strip width
  static const double xs = 4.0;

  /// 8.0 — Compact list item vertical padding, inline element gaps
  static const double sm = 8.0;

  /// 16.0 — Default horizontal padding on list screens, form field spacing
  static const double md = 16.0;

  /// 24.0 — Section padding, form horizontal padding, empty-state text padding
  static const double lg = 24.0;

  /// 32.0 — Layout gaps between major sections
  static const double xl = 32.0;
}
```

For each of the 9 files listed below, do two things:

1. **Add import** at top of file: `import 'package:vida_ativa/core/theme/app_spacing.dart';`
2. **Replace literal values** in `EdgeInsets` and `SizedBox` constructs per the mapping below.

**Substitution rules:**
- `EdgeInsets.all(4)` or `EdgeInsets.all(4.0)` -> `EdgeInsets.all(AppSpacing.xs)`
- `EdgeInsets.all(8)` or `EdgeInsets.all(8.0)` -> `EdgeInsets.all(AppSpacing.sm)`
- `EdgeInsets.all(16)` or `EdgeInsets.all(16.0)` -> `EdgeInsets.all(AppSpacing.md)`
- `EdgeInsets.all(24)` or `EdgeInsets.all(24.0)` -> `EdgeInsets.all(AppSpacing.lg)`
- `EdgeInsets.all(32)` or `EdgeInsets.all(32.0)` -> `EdgeInsets.all(AppSpacing.xl)`
- Same for `EdgeInsets.symmetric(horizontal: N)`, `EdgeInsets.symmetric(vertical: N)`, `EdgeInsets.symmetric(horizontal: N, vertical: M)`
- `SizedBox(height: 4)` -> `SizedBox(height: AppSpacing.xs)`, etc.
- `SizedBox(width: 4)` -> `SizedBox(width: AppSpacing.xs)`, etc.

**DO NOT replace:**
- Values that don't match the scale: 6, 12, 72, 900 (durations), etc.
- `BorderRadius.circular(N)` values — these are NOT spacing tokens
- `Container(width: 4)` that represents a color strip (visual element, not spacing)
- Font sizes (fontSize: 16 is NOT a spacing value)
- Duration values
- Dimension values inside `Container(height: 72)` — these are component sizes, not spacing
