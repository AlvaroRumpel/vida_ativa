## Interaction States

### Week Navigation
- **Initial state:** Current week displayed (e.g., "Semana de 31 mar–6 abr")
- **Previous week:** ← button enabled (disabled only if no past weeks to show, per business logic)
- **Next week:** → button enabled (disabled only if no future weeks to show, per business logic)
- **Visual feedback:** Button press → light color change (Material 3 standard)

### Day Chips
- **Unselected:** Background #F0EDE8, text #4A4A4A, weight 500, border 0.5px #CFC5B0
- **Selected:** Background primaryGreen (#7B5D0A), text white, weight 700, no border

### Booking Detail Sheet
- **Opening:** showModalBottomSheet with isScrollControlled: true
- **Sheet backdrop:** Default Material (semi-transparent dark overlay, dismissible on tap outside)
- **Pending booking:** Both Confirm and Reject buttons visible (full width, 52px height each)
- **Confirmed/Rejected booking:** Buttons hidden, detail view only
- **Loading on confirm/reject:** Button disabled, circular progress shows
- **Error on confirm/reject:** Error message displayed inline (red, 12px), buttons re-enable
