For the timeline bar use `LayoutBuilder` to get the available width, then compute `left` and `width` in pixels:
```dart
LayoutBuilder(builder: (ctx, constraints) {
  final total = constraints.maxWidth;
  final left = (draft.fromHour / 24) * total;
  final width = ((draft.toHour - draft.fromHour) / 24) * total;
  return Stack(children: [
    Container(height: 3, color: AppTheme.lineHair),
    Positioned(left: left, width: width, top: 0, bottom: 0,
      child: Container(color: AppTheme.orange)),
  ]);
})
```

Day label helper — convert `daysOfWeek` list to a readable string:
- If all weekdays [1,2,3,4,5]: "SEG–SEX"
- If [6,7]: "SÁB·DOM"
- If [1,2,3,4,5,6,7] or empty: "TODOS"
- Otherwise: join abbreviations e.g. "SEG·QUA·SEX"

Day abbreviations: ['SEG','TER','QUA','QUI','SEX','SÁB','DOM'] (index = daysOfWeek value - 1).

Tier label format: `'FAIXA ${(index+1).toString().padLeft(2,'0')} · $daysLabel'`
