Create `lib/features/schedule/cubit/schedule_cubit.dart` — the core data logic for the schedule feature.

**Constructor:**
```dart
class ScheduleCubit extends Cubit<ScheduleState> {
  final FirebaseFirestore _firestore;
  final AuthCubit _authCubit;

  StreamSubscription<QuerySnapshot>? _slotsSubscription;
  StreamSubscription<QuerySnapshot>? _bookingsSubscription;
  StreamSubscription<DocumentSnapshot>? _blockedDateSubscription;

  // Cache last values from each stream for recomputation
  List<SlotModel>? _cachedSlots;
  List<BookingModel>? _cachedBookings;
  bool _cachedIsBlocked = false;
  DateTime? _selectedDate;

  ScheduleCubit({
    required FirebaseFirestore firestore,
    required AuthCubit authCubit,
  })  : _firestore = firestore,
        _authCubit = authCubit,
        super(const ScheduleInitial());
```

**`selectDay(DateTime date)` method:**
1. Store `_selectedDate = date`
2. Call `_cancelSubscriptions()` to cancel all three previous subscriptions
3. Reset cache fields (`_cachedSlots = null`, `_cachedBookings = null`, `_cachedIsBlocked = false`)
4. Emit `ScheduleLoading()`
5. Compute `dateString` as `"${date.year}-${date.month.toString().padLeft(2, '0')}-${date.day.toString().padLeft(2, '0')}"`
6. Compute `weekday = date.weekday` (1=Mon..7=Sun, matches SlotModel.dayOfWeek directly)
7. Start three Firestore stream subscriptions:

**Stream 1 — Active slots for this weekday:**
```dart
_slotsSubscription = _firestore
    .collection('slots')
    .where('isActive', isEqualTo: true)
    .where('dayOfWeek', isEqualTo: weekday)
    .snapshots()
    .listen((snapshot) {
      _cachedSlots = snapshot.docs.map((d) => SlotModel.fromFirestore(d)).toList();
      _recompute();
    }, onError: (e) => emit(ScheduleError('Erro ao carregar horários.')));
```

**Stream 2 — Non-cancelled bookings for this date:**
```dart
_bookingsSubscription = _firestore
    .collection('bookings')
    .where('date', isEqualTo: dateString)
    .where('status', whereIn: ['pending', 'confirmed'])
    .snapshots()
    .listen((snapshot) {
      _cachedBookings = snapshot.docs.map((d) => BookingModel.fromFirestore(d)).toList();
      _recompute();
    }, onError: (e) => emit(ScheduleError('Erro ao carregar reservas.')));
```

**Stream 3 — Blocked date check (single document):**
```dart
_blockedDateSubscription = _firestore
    .collection('blockedDates')
    .doc(dateString)
    .snapshots()
    .listen((snapshot) {
      _cachedIsBlocked = snapshot.exists;
      _recompute();
    }, onError: (e) => emit(ScheduleError('Erro ao verificar bloqueios.')));
```

**`_recompute()` method:**
- Guard: if `_cachedSlots == null || _cachedBookings == null || _selectedDate == null` return (wait for all streams)
- If `_cachedIsBlocked` is true: emit `ScheduleLoaded(slots: [], selectedDate: _selectedDate!, isBlocked: true)` and return
- Get `currentUserId`: read `_authCubit.state` — if `AuthAuthenticated`, use `state.user.uid`; otherwise use empty string `''`
- Build `List<SlotViewModel>` by mapping each cached SlotModel through `_resolveStatus()`:

```dart
SlotStatus _resolveStatus(SlotModel slot, String dateString, String currentUserId) {
  final booking = _cachedBookings!.cast<BookingModel?>().firstWhere(
    (b) => b!.slotId == slot.id,
    orElse: () => null,
  );
  if (booking == null) return SlotStatus.available;
  if (booking.userId == currentUserId) return SlotStatus.myBooking;
  return SlotStatus.booked;
}
```

- Sort the resulting list by `slot.startTime` (lexicographic sort works for "HH:mm" format)
- Emit `ScheduleLoaded(slots: sortedList, selectedDate: _selectedDate!, isBlocked: false)`

**`_cancelSubscriptions()` method:**
```dart
void _cancelSubscriptions() {
  _slotsSubscription?.cancel();
  _slotsSubscription = null;
  _bookingsSubscription?.cancel();
  _bookingsSubscription = null;
  _blockedDateSubscription?.cancel();
  _blockedDateSubscription = null;
}
```

**`close()` override:**
```dart
@override
Future<void> close() {
  _cancelSubscriptions();
  return super.close();
}
```
