  - `IconButton(icon: Icon(Icons.chevron_right), onPressed: onNextWeek)` — null disables it

- Week label function:
  ```dart
  String _weekLabel(DateTime weekStart) {
    final weekEnd = weekStart.add(const Duration(days: 6));
    const months = ['Jan','Fev','Mar','Abr','Mai','Jun','Jul','Ago','Set','Out','Nov','Dez'];
    if (weekStart.month == weekEnd.month) {
      return 'Semana de ${weekStart.day}\u2013${weekEnd.day} ${months[weekEnd.month - 1]}';
    }
    return 'Semana de ${weekStart.day} ${months[weekStart.month - 1]}\u2013${weekEnd.day} ${months[weekEnd.month - 1]}';
  }
  ```
  The `\u2013` is an en-dash. Output example: "Semana de 17-23 Mar" (with en-dash between numbers).

- Wrap in `Padding(padding: EdgeInsets.symmetric(vertical: 8))`

**File 6: `lib/features/schedule/ui/schedule_screen.dart`**

Root screen that owns week navigation state and connects to ScheduleCubit.

- `ScheduleScreen` is a `StatefulWidget`
- State fields:
  ```dart
  late DateTime _weekStart;
  late DateTime _selectedDay;
  late DateTime _currentWeekMonday;
  ```

- `initState()`:
  ```dart
  final now = DateTime.now();
  _currentWeekMonday = DateTime(now.year, now.month, now.day)
      .subtract(Duration(days: now.weekday - 1));
  _weekStart = _currentWeekMonday;
  _selectedDay = DateTime(now.year, now.month, now.day);
  // Trigger initial data load
  WidgetsBinding.instance.addPostFrameCallback((_) {
    context.read<ScheduleCubit>().selectDay(_selectedDay);
  });
  ```

- 8-week limit calculation:
  ```dart
  DateTime get _maxWeekStart => _currentWeekMonday.add(const Duration(days: 7 * 7)); // 7 jumps = 8 weeks
  bool get _canGoPrevious => _weekStart.isAfter(_currentWeekMonday);
  bool get _canGoNext => _weekStart.isBefore(_maxWeekStart);
  ```

- Week navigation handlers:
  ```dart
  void _goToPreviousWeek() {
    setState(() {
      _weekStart = _weekStart.subtract(const Duration(days: 7));
      _selectedDay = _weekStart; // Select Monday of the new week
    });
    context.read<ScheduleCubit>().selectDay(_selectedDay);
  }

  void _goToNextWeek() {
    setState(() {
      _weekStart = _weekStart.add(const Duration(days: 7));
      _selectedDay = _weekStart; // Select Monday of the new week
    });
    context.read<ScheduleCubit>().selectDay(_selectedDay);
  }
  ```

- Day selection handler:
  ```dart
  void _onDaySelected(DateTime day) {
    setState(() => _selectedDay = day);
    context.read<ScheduleCubit>().selectDay(day);
  }
  ```

- `build()` method:
  ```dart
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Agenda')),
      body: Column(
        children: [
          WeekHeader(
            weekStart: _weekStart,
            onPreviousWeek: _canGoPrevious ? _goToPreviousWeek : null,
            onNextWeek: _canGoNext ? _goToNextWeek : null,
          ),
          DayChipRow(
            weekStart: _weekStart,
            selectedDay: _selectedDay,
            onDaySelected: _onDaySelected,
          ),
          const SizedBox(height: 8),
          Expanded(
            child: BlocBuilder<ScheduleCubit, ScheduleState>(
              builder: (context, state) => SlotList(state: state),
            ),
          ),
        ],
      ),
    );
  }
  ```
